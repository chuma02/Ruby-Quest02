# Welcome to Ruby Quest02
***

## Task
The goal of this project is to practice basic Ruby programming concepts including:

Conditional statements with multiple conditions (if, else).

Functions (methods) and parameters.

Loops (while) and iteration.

Returning values from functions.

The challenge is to correctly implement these concepts while handling multiple conditions, function parameters, and
 loop control in Ruby.

## Description
This project contains five exercises:

My First If Multiple Conditions

Implemented multiple conditions in if statements to print the correct messages.

Example: a > b && a < c && a == d.

My First Function

Created a simple Ruby function my_first_function that prints a string when called.

My First While

Used a while loop to print "I want to code" 100 times, incrementing an index.

My First Param Function

Created a function that accepts an integer parameter and prints "detonation in... X seconds".

Implemented a while loop to count down from 10 to 1.

My First Return Function

Created a function my_get_seven that returns the number 7.
